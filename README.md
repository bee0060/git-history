## Git History

This is a tool to view git history

##Requirements

* WebPack
* Mithril.js
* Express
* Ramda.js
* Babel
* [Css Modules](http://glenmaddern.com/articles/css-modules)
* Brace 

## Run

```
npm install
npm start
open http://127.0.0.1:3000
```
